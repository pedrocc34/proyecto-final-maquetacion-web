Formularios

Etiquetas Input:

Se utilizan para crear campos de entrada, como cajas de texto, contraseñas, botones de radio, casillas de verificación, etc.

Etiquetas textarea:

Se utliizan para crear áreas de texto de varias líneas en un formulario. A diferencia de la etiqueta "input type=text", que crea una caja de texto de una sola línea, textarea permite a los usuarios ingresar y editar texto en un espacio más grande.



Etiquetas select y action:

Se utilizan para crear listas desplegables (también conocidas como menús desplegables o selectores) en formularios web. Estas etiquetas permiten a los usuarios seleccionar una opción de una lista predefinida.



Botón "submit":

El elemento clave para enviar un formulario es el botón de tipo "submit". Puedes usar la etiqueta "button" o la etiqueta "input" con el atributo type="submit".




Métodos de envío:

  1. GET
    • Los datos del formulario se adjuntan a la URL como parámetro.
    • Es visible en la barra de direcciones del navegador.
    • Tiene limitaciones en la cantidad de datos que se pueden enviar.
    • Es menos seguro para información sensible como contraseñas, ya que los datos son visibles en la URL.

  2. POST
    • Los datos del formulario se envían en el cuerpo de la solicitud HTTP, no en la URL.
    • No es visible en la barra de direcciones.
    • Puede manejar cantidades más grandes de datos.
    • Es más seguro para información sensible, ya que los datos no son visibles en la URL.

En resumen, el método GET se utiliza generalmente para solicitudes de búsqueda o solicitudes donde la información no es sensible y puede ser visible en la URL. El método POST se utiliza para enviar datos sensibles o cuando la cantidad de datos es grande y no se desea mostrar la URL.
